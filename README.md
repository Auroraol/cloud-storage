# 前端

```shell
# 启动frp客户端
cd frp_0.61.1_windows_amd64
# cmd 
frpc -c ./frpc.toml
```

```shell
# 进入项目目录
cd cloud-storage-front-end

# 安装依赖
pnpm i

# 启动服务
pnpm dev
```
